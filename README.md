# ChairMatch

Dein Beauty-Partner in ganz Deutschland.

## About

ChairMatch is a premium beauty and salon booking platform for Germany. It connects customers with top-rated hairstylists, barbers, and beauty professionals nationwide.

## Features

- Browse and discover salons and stylists near you
- - Book appointments with real-time availability
  - - Premium gold-themed UI with dark luxury aesthetic
    - - Mobile-first responsive design
      - - Rating and review system
        - - Special offers and promotions
         
          - ## Tech Stack
         
          - - HTML5 / CSS3 / Vanilla JavaScript
            - - Single-page application (SPA)
              - - Progressive Web App (PWA) ready
                - - Google Fonts (Cormorant Garamond, Cinzel, DM Sans)
                 
                  - ## Getting Started
                 
                  - Simply open `index.html` in your browser to run the app locally.
